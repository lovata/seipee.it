Sync Settings Help

What does this sync do?

This automatic synchronization performs two important tasks:

  1. Sync undelivered orders - Fetches all orders from Seipee API where DocEvaso = false (not delivered) and creates/updates them in the system
  2. Update delivery status - Checks all local orders marked as undelivered and updates their status if they became delivered in Seipee

How to configure the interval

  • Hours: Set a value from 0 to 23
  • Minutes: Set a value from 0 to 59

Examples

Hours Minutes Result
4 0 Runs every 4 hours (00:00, 04:00, 08:00, 12:00, 16:00, 20:00)
1 30 Runs every 1 hour and 30 minutes
0 30 Runs every 30 minutes
0 15 Runs every 15 minutes

Important Notes

  • Cron must be configured: Make sure your system cron is set up to run Laravel's scheduler
  • Performance: Very frequent syncs (e.g., every few minutes) may impact system performance
  • Recommended: For most cases, syncing every 2-4 hours is sufficient
